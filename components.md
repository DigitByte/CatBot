# CatBot components

This list covers the core hardware used in the CatBot build. Use it as a reference for labs, prototypes, or a full build.

# Core components

| Component | Quantity | Purpose |
| --- | --- | --- |
| Brushless servomotor | 12 | Leg actuators |
| 25T disk | 8 | Hip joint coupling (hip1, hip2) |
| GT2 pulley, 36T | 4 | Knee joint transmission |
| Servo heatsink | 2 | Knee joint thermal management |
| 7mm bearing | 20 | Belt tensioning and knee support |
| 37mm bearing | 8 | Hip joint support |
| hardware printing threads | 16 | Bearing retention |
| GT2 belt, 30cm | 4 | Knee belt drive |
| PLA filament, white 1kg | 1 | Main prints |
| PLA filament, black 1kg | 1 | Accent prints |
| DC-DC converter 10A | 1 | Servo power regulation |
| Teensy 4.0 | 1 | Servo control and sensor interface |
| 5V relay | 1 | Servo power switching |
| 5V 3.2A regulator | 1 | Power for Raspberry Pi |
| Power switch | 1 | Main power cutoff |
| Rubber feet pads | 4 | Traction and vibration damping |
| Battery | 1 | System power |
| Raspberry Pi 4 | 1 | Onboard computer |
| Raspberry Pi cooling kit | 1 | Thermal management |
| BNO080 IMU | 1 | Motion and orientation sensing |
| Hall effect sensor | 4 | Foot contact sensing |

# Price and sourcing
Check the price list in `price_list.ods` for vendor examples. A full build is roughly 1000 EUR, with the motors as the largest cost driver. You can substitute lower-cost motors for prototypes, but performance may vary.
